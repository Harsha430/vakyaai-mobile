# VākyaAI: The Architect of Eloquence 🌌

VākyaAI is a premium AI-powered pitch assistant designed to transform raw ideas into boardroom-ready narratives. Built with a "Mystical Glassmorphism" design philosophy, it combines cutting-edge psychometric analysis with advanced user experience.

---

## ✨ Key Features

### 🛡️ Secure Authentication
- **JWT-Locked Sessions**: Secure Login and Signup flow with persistent session management.
- **Secure Storage**: Encrypted token storage for user privacy.

### 📊 Command Center (4-Tab Dashboard)
- **Lab 1: Insights**: Radar charts visualizing 6 key performance metrics (Impact, Strategy, Technical Depth, etc.).
- **Lab 2: Optimization**: AI-refined pitch generation with a "Metadata Factory" for social hooks and emails.
- **Lab 3: Deck Architect**: Logical slide structure widgets to build your narrative arc.
- **Lab 4: Growth**: Personalized practice roadmap and a Q&A defense trainer.

### 🎙️ Pitch Studio
- **Voice Intelligence**: Hands-free pitch composition using high-accuracy speech-to-text.
- **Audience Context**: Tailored analysis for Investors, Technical Juries, or Accelerators.
- **Dynamic Templates**: Pre-configured frames for Hackathons, Seed Rounds, and Academic Vivas.

---

## 🎨 Design Philosophy: Mystical Glassmorphism

VākyaAI isn't just a tool; it's an experience.
- **Rich Aesthetics**: Vibrant gradients, grain textures, and backdrop blurs.
- **Typography**: A blend of `Cinzel` (Elegance) and `Inter` (Clarity).
- **Kinetic UI**: Staggered animations and smooth state transitions powered by `flutter_animate`.

---

## 🚀 Tech Stack

- **Framework**: Flutter (Dart)
- **State Management**: Flutter Riverpod
- **Intelligence**: Gemini AI (via Rest API)
- **Networking**: Http with Authenticated Interceptors
- **Persistence**: Flutter Secure Storage
- **UI Components**: Glassmorphism, Fl Chart, Google Fonts

## 🛠️ Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Harsha430/vakyaai-mobile.git
   cd vakyaai-mobile
   ```

2. **Install dependencies**:
   ```bash
   flutter pub get
   ```

3. **Configure API**:
   Update the `baseUrl` in `lib/core/constants.dart` with your VākyaAI backend endpoint.

4. **Run the app**:
   ```bash
   flutter run
   ```
